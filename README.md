# near-realtime-databricks
Repo was created to showcase Near Real-Time databricks project using SDP (lakeflow spark declarative pipelines),
what data should be processed (using streaming tables and materialized views) rather than how,
automatically handling orchestration and optimization. SDP is designed for incremental ingestion from cloud storage and message buses.
